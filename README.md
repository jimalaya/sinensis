Sinensis is a wellness app, focused on handing the user diverse activities to reduce stress. It is not meant to replace psychological treatment, but it offers day to day solutions for anyone suffering from anxiety and depression. 

It was developed in Android Studio, particularly in Java, with XML features.

It also features a calendar, and the user has the option to schedule certain activities, and synchronize them with the outer calendar of the mobile. Aditionally, some of them open up google maps and search for a particular recommended place in the nearby area where you can do the activity.
